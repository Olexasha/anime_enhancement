# Anime Enhancement GUI

Современный графический интерфейс для улучшения аниме-видео с использованием нейронных сетей (Waifu2x, Real-ESRGAN, RIFE).

## 🚀 Возможности

- **Современный интерфейс** - Красивый и интуитивный GUI на базе PySide6
- **Управление настройками** - Легкое редактирование всех параметров проекта
- **Настройка нейронных сетей** - Конфигурация Waifu2x, Real-ESRGAN и RIFE с рекомендациями
- **Мониторинг процесса** - Стриминг логов в реальном времени с цветовой подсветкой
- **Безопасная конфигурация** - Сохранение настроек в override файл с возможностью отката
- **Системные рекомендации** - Предупреждения о VRAM и оптимальных настройках

## 📋 Требования

- Python 3.12+
- PySide6
- psutil (для мониторинга системы)
- Существующий проект anime_enhancement

## 🛠️ Установка

### 1. Установка зависимостей

```bash
# Установка зависимостей для GUI
pip install -r requirements-gui.txt

# Или установка вручную
pip install PySide6>=6.6.0 psutil>=5.9.0
```

### 2. Подготовка проекта

Убедитесь, что у вас есть:
- Файл `main.py` в корневой директории проекта
- Файл `src/config/settings.py` с настройками
- Структура папок `data/input_video/`, `data/output_video/` и т.д.

### 3. Запуск GUI

```bash
# Из корневой директории проекта
python gui/app.py
```

## 🎯 Использование

### Основные функции

1. **Project Info** - Информация о проекте и статус файлов
2. **Settings** - Настройки путей и параметров обработки
3. **Neural Networks** - Конфигурация нейронных сетей
4. **Logs** - Мониторинг процесса обработки

### Настройка проекта

1. **Выбор входного видео**:
   - Нажмите "Browse..." рядом с "Input Video"
   - Выберите видеофайл в формате MP4, MKV, AVI и др.

2. **Настройка путей**:
   - Укажите папки для аудио, временных файлов, выходного видео
   - Пути к AI инструментам определяются автоматически

3. **Параметры обработки**:
   - Выберите формат выходных изображений (PNG, JPG, BMP, TIFF)
   - Установите диапазон батчей для обработки
   - Выберите целевое разрешение (1080p, 2K, 4K, 8K)

### Настройка нейронных сетей

#### Waifu2x (Денойз)
- **Модель**: cunet (рекомендуется для аниме), upconv_7_anime_style_art_rgb, upconv_7_photo
- **Уровень шума**: 0-3 (3 для сильного шума)
- **Масштабирование**: 1x-4x
- **TTA режим**: Улучшает качество, но замедляет в 8 раз

#### Real-ESRGAN (Апскейл)
- **Модель**: realesr-animevideov3 (рекомендуется для аниме)
- **Масштабирование**: 2x-4x
- **TTA режим**: Улучшает качество, но замедляет в 8 раз

#### RIFE (Интерполяция)
- **Модель**: rife-v4.6 (последняя), rife-anime (оптимизирована для аниме)
- **Множитель кадров**: 2x-8x (2x = 60fps, 4x = 120fps, 8x = 240fps)
- **UHD режим**: Рекомендуется для 4K и выше
- **TTA режимы**: Spatial и Temporal для улучшения качества
- **Потоки**: Настройка load:proc:save потоков

### Рекомендации и предупреждения

GUI автоматически анализирует ваши настройки и показывает:
- ⚠️ **Предупреждения** о потенциальных проблемах
- 💡 **Рекомендации** по оптимизации
- 🔧 **Кнопку "Apply Recommended Settings"** для применения безопасных настроек

### Запуск обработки

1. Нажмите **"▶️ Start Processing"**
2. Следите за прогрессом в панели **Logs**
3. Используйте **"⏹️ Stop"** для остановки процесса
4. **"⏸️ Pause"** для приостановки (если поддерживается)

### Управление логами

- **Цветовая подсветка**: ERROR (красный), WARNING (оранжевый), SUCCESS (зеленый), INFO (синий)
- **Автоскролл**: Автоматическая прокрутка к новым сообщениям
- **Экспорт**: Сохранение логов в текстовый файл
- **Очистка**: Удаление всех логов

## ⚙️ Конфигурация

### Сохранение настроек

GUI сохраняет изменения в `settings_gui_override.json` рядом с проектом. Это позволяет:
- Не изменять оригинальный `settings.py`
- Легко откатить изменения
- Создавать разные профили настроек

### Откат изменений

1. Удалите файл `settings_gui_override.json`
2. Перезапустите GUI
3. Настройки вернутся к значениям по умолчанию

### Резервные копии

При записи в `settings.py` (если включена) создается резервная копия в папке `backups/` с временной меткой.

## 🐛 Устранение неполадок

### GUI не запускается

1. Проверьте установку PySide6: `pip list | grep PySide6`
2. Убедитесь, что запускаете из корневой директории проекта
3. Проверьте наличие файлов `main.py` и `src/config/settings.py`

### Процесс не запускается

1. Проверьте путь к входному видео
2. Убедитесь, что папки для выходных файлов существуют
3. Проверьте свободное место на диске (минимум 10 ГБ)

### Ошибки конфигурации

1. Удалите `settings_gui_override.json` для сброса к умолчанию
2. Проверьте права доступа к файлам
3. Убедитесь, что `settings.py` не поврежден

### Проблемы с производительностью

1. Используйте кнопку "Apply Recommended Settings"
2. Уменьшите количество потоков в RIFE
3. Отключите TTA режимы для ускорения
4. Проверьте использование VRAM

## 📝 Логирование

Логи GUI сохраняются в папке `logs/` с именем файла `gui_YYYYMMDD.log`.

Уровни логирования:
- **DEBUG** - Отладочная информация
- **INFO** - Общая информация
- **WARNING** - Предупреждения
- **ERROR** - Ошибки
- **SUCCESS** - Успешные операции

## 🤝 Поддержка

При возникновении проблем:
1. Проверьте логи в папке `logs/`
2. Убедитесь, что все зависимости установлены
3. Проверьте совместимость версий Python и PySide6
